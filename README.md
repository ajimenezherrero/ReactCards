# DroneClassProject

The goal of this project is create a cards system with [React](https://reactjs.org/).


# Documentation

## Getting Started

### Install all required packages:
```
npm install
```

### Run the application:
```
npm start
```

### Other Scripts

```bash
npm test # run testing
```

## Technologies / Libraries

- [React](https://reactjs.org/)
